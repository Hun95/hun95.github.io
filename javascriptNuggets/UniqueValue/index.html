🚀 Map Function

🔨 특정한 값만 추출하기

index.js
js
const people = [
  {
    name: "Lady gaga",
    age: 32,
    job: "Singer",
  },
  {
    name: "Post Malon",
    age: 28,
    job: "Singer",
  },
  {
    name: "John Sina",
    age: 40,
    job: "Wrestler",
  },
  {
    name: "Lee",
    age: 60,
    job: "President",
  },
];

Map 함수에서 사용한 데이터를 그대로 가져왔습니다.
데이터 내에 직업은 Singer, Wrestler, President입니다. 레이디 가가와 포스트말론의 직업이 겹칩니다.

map 함수를 이용해 job을 출력한다면, Singer가 두 번 출력됩니다.
중복을 제거한 후 출력을 하고 싶을 때는 어떻게 할까요?

js
const jobs = new Set(people.map(item => item.job));
console
{"Singer", "Wrestler", "President"}

map 함수를 사용했을 때는 [] 배열로 값을 전달받았는데, 지금은 {} 객체로 값을 돌려받았습니다.

배열로 값을 받고싶다면 어떻게 할까요??

js
const jobs = [...new Set(people.map(item => item.job))];
console
["Singer", "Wrestler", "President"]

중복된 값을 없앴고 배열로 돌려받았습니다. 그러면 이제 출력을 해야겠죠?

html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>asd</title>
  </head>
  <body>
    <div class="asd"></div>
  </body>
</html>
<script>
  const people = [
    {
      name: "Lady gaga",
      age: 32,
      job: "Singer",
    },
    {
      name: "Post Malon",
      age: 28,
      job: "Singer",
    },
    {
      name: "John Sina",
      age: 40,
      job: "Wrestler",
    },
    {
      name: "Lee",
      age: 60,
      job: "President",
    },
  ];
  const jobs = [...new Set(people.map(item => item.job))];
  console.log(jobs);
  const list = document.querySelector(".asd");
  list.innerHTML = jobs
    .map(listitem => {
      return `<h1>${listitem}</h1>`;
    })
    .join("");
</script>

html에 그대로 작성해보면 정상적으로 출력됨을 확인하실 수 있습니다.
